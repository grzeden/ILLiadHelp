Description:

This queue contains requests whose PDF files have been received or scanned, and placed on the ILLiad FTP server for patron access.

To process:

None.

More information:

If a request in this queue does not exist on the ILLiad FTP server it will need to be routed to the Request Finished queue if it has been accessed by the patron, or requested again if it has not been accessed.

This information has been updated for ILLiad 8.5.2.0.

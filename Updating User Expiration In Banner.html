Updating User Expiration In Banner

Dennis Grzegorzewski, 18 May 2015 (created 18 May 2015).

UNFINISHED.

Description:

Requests in this status queue have been placed by users whose expiration date has been passed.

To process:

Look up patron in Voyager.

Update their expiration as indicated.

Close request.

It will be routed to Awaiting Citation Completion

More information:

When requests are routed to this queue they are immediately routed to the Awaiting User Clearance status queue if the user has not been cleared, or to the Awaiting Citation Completion status queue if the users expiration date has not passed.

This information has been updated for ILLiad 8.5.2.0.

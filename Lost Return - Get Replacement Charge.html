Description:

This status is currently unused. If there are any requests in this queue they have been placed here by mistake and need to be moved.

To process:

  1. Route this request to its previous status.
    1. Click on the History tab.
    2. In the Tracking section scroll down to the bottom of the screen.
    3. Find the second oldest status in the Changed To column.
    4. Route the request to this status.
  3. Or route this request to Awaiting Extensive Searching, with a note, for futher processing.

Awaiting Renewal OK Processing

Dennis Grzegorzewski, 18 May 2015 (created 18 May 2015).

COMPLETED.

Description:

This queue holds requests that have the OCLC Category RenewalOk. The lender has granted a new due date for the requests in this queue.

These requests need to have their due dates updated, and emails need to be sent to the customers to inform them that their requests have been renewed and of their new due dates.

To process:

  1. Read the Renewal note in the Note section for the new Due Date.
  2. Subtract 5 days from this date to create the new due date.
    1. If this date is earlier than tomorrow, use tomorrow's date.
    2. If this date seems very old, see the instructions below to read the OCLC request for the due date.
  3. Enter this new date into the Due Date field of the Item Information section of the Borrowing Request window.
  4. Click on the Save Changes icon.
  5. Click on the Send EMail icon in the User section.
  6. Click on Renewal OK in the drop-down menu.
    1. This opens the Edit EMail Form page.
  7. Click on the Send button.
  8. The request's status is automatically changed to Checked Out to Customer.
  9. The Borrowing Request window automatically closes.
  10. Repeat until all requests are processed.

More information:

If for some reason there is no Renewal note, or if the due date is really old, the new due date can be found in the OCLC request by doing the following from the Borrowing Request window.

  1. Click on the OCLC ribbon tab, to open the OCLC icon ribbon.
  2. Click on the Show icon in the Request section.
    1. This opens the OCLC Request window.
    2. Read the New Due Date.
    3. Close this window.
  3. Again, if this date is earlier than tomorrow, use tomorrow's date.

This information has been updated for ILLiad 8.5.2.0.

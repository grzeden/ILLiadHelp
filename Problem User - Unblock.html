Problem User - Unblock

Dennis Grzegorzewski, 18 May 2015 (created 18 May 2015).

UNFINISHED.

Description:

This status holds requests for overdue items that have been returned by the user. Requests are automatically routed to this status when returns are printed and the user has been blocked for this item. Users need to be unblocked if all blocking items have been returned.

To process:

  1. Open a request in the Problem User - Unblock status queue.
  2. Click on the View icon in the User section of the Borrowing Process ribbon tab.
    1. This opens the User window.
  4. Click on the Requests tab.
  5. Sort the request list by the Due Date field.
  6. If there are no other other items checked out to the customer that are a week overdue and have the Last Overdue Notice Sent equal to 3.
    1. Then click on the Cleared icon.
  8. Close the User window.
  9. Click on the Route icon in the Routing section of the Borrowing Process ribbon tab.
  10. Select Request Finished from the dropdown menu.
  11. Move on to the next request in the queue.

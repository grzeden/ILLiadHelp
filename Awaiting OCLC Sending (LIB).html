Awaiting Request Processing (LIB)

Dennis Grzegorzewski, 18 May 2015 (created 18 May 2015).

UNFINISHED.

Description:

Requests in this status need to be verified for correct citation and then sent to possible lenders. The main difference between requests in this status and those in 'Awaiting Request Processing (Checked)' are that these requests are made by library workers. Library workers are allowed to request textbooks, while other students cannot.

To process:

  1. Double-click "Awaiting Request Processing" under the Main>Borrowing Tab
  2. Choose a request to process
  3. Verify that the citation (ISSN/ISBN, Article Title, Volume, Issue, Pg. #, Edition, etc) is correct using 'Google Search'
    1. Click the 'Google Search' Tab'
    2. Search for sources that can contain the correct citation. ILLiad automatically inputs the Book Title and Article Title into Google. If the Google search does not find good sources, include Author and/or Journal Title, if applicable, into the search.
    3. Check the citations with the source, then add information or make any changes, if necessary
  4. Search for the book/article in OCLC
    1. Click the 'OCLC' Tab
    2. Click the magnifying glass by ISxN to search by the ISBN/ISSN
    3. Double click the record that has the most holdings but is not an internet resource
  5. Fill in the request's workform with lenders
    1. For journals, double click the lenders that indicate they have the volume for the request
    2. For books, a more extensive process is required
      1. Log on to FirstSearch (URL: http://firstsearch.oclc.org/ Auth.: 100077549 PW: pmk)
      2. Paste the ISBN of the request into the 'Search for:' field and select 'ISBN' in the drop box next to the field
      3. Click Search
      4. Choose the book that matches the request and verify the edition
      5. Under 'Custom Holdings Path', select 'BOOK' and click search
      6. Starting from the top of the list, check to see if each lender has the requested book available
        1. Click a lender name
        2. Read the book's entry and ensure that the book is available. The book is not available if the book is located in reserve, or has a due date
      7. For every lender that has the requested book available, select their lender code on ILLiad's workform. Keep searching for possible lenders until 5 lenders are selected
  6. Click 'Update Work Form'
  7. Click 'Send Request'
  8. Click 'Request Sent'

More information:

This information has been updated for ILLiad 8.5.2.0.

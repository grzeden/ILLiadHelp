Awaiting Return Label Printing

Dennis Grzegorzewski, 24 November 2014 (created 3 November 2014).

UNFINISHED.

Description:

This queue holds requests that have been checked in from the customer at the Circulation Desk, or that have been pulled from the Hold Shelf because they are overdue. These items need to have shipping labels printed so that they may be returned to the lender. The items to be returned should be on the Awaiting Return Label Printing shelf.

To process:

  1. Click on the Main (F12) tab.
  2. Click on the Borrowing ribbon tab.
  3. Click on the Print Returns icon in the Returns section.
  4. If this opens the ILLiad Printer Settings (BorrowingReturnSlips.doc) dialog window.
    1. Select ILL Printer and click on the Print button.
    2. This closes the dialog window.
  5. If this opens the Word Template BorrowingReturnSlips.doc, and a Word document with the requests already merged.
    1. On the merged Word document click on the Print icon.
      1. This will print out the Borrowing Return Slips.
    2. Close both Word documents.
    3. When asked to save changes Click on the Don't Save button.
  6. Get the Borrowing Return Slips from the printer.
  7. Match the printouts with the items on the Awaiting Return Label Printing shelf and place inside.
    1. If there are items that have no corresponding printouts, then give these items to Dennis.
    2. If there are printouts that have no corresponding items to return, then give these printouts to Dennis.
    3. If there are printouts that do not have shipping addresses, then give these items and printouts to Dennis.
  8. Move the items to ship to the Ready To Ship shelf.
    1. Out of state items are placed to the left of the red marker.
    2. ILDS items are placed to the right of the red marker.

More information:

When no shipping address prints it is because no lender was sellected for the item on receipt or that a new lender was given no address information.

This information has been updated for ILLiad 8.5.2.0.

Awaiting Odyssey Processing

Dennis Grzegorzewski, 18 May 2015 (created 18 May 2015).

COMPLETED.

Description:

This queue holds the list of article requests that are ready to be received through Odyssey Electronic Delivery. The files that are to be received need to be checked for correctness, completeness, and readability before they are delivered.

To process:

  1. Go to the ILLiad Client window.
  2. Click on the Refresh (F5) icon.
    1. This refreshes the listed data to the current state.
  3. Click on the Main (F12) tab.
  4. Click on the Borrowing tab in the ribbon menu.
    1. This displays the list of Borrowing Request queues.
  5. Click on the Electronic Delivery icon in the Receive Request section.
    1. This opens the Electronic Delivery window
    2. If there are requests remaining to be processed, these must be cleared out before new requests can be processed.
  6. Click on the Process Files icon to load in new requests to receive.
  7. Requests that load into the No ILL Number Match section need to be matched to the request's ILL or Transaction Number.
    1. Copy the request's ILL number from the cover page. If the request's ILL number cannot be found in the document, an extensive search is required
      1. Look for the article title and copy the title
      2. Search for a request that matches the article title
        1. On the main ILLiad window, click on the Home tab in the ribbon menu
        2. Paste the article title into Citation Information
        3. Click Search
      3. Verify that the request is the one being processed (check the rest of the citation information)
      4. Change the request status to Request Sent
      5. Copy the ILL Number
      6. Close the request
    2. On the Electronic Delivery window, paste the ILL Number into the ILL field
    3. Click Match ILL Number
  8. Requests that load into the No Ariel Lender Match section need to be matched to one of the Lenders in the request's Lender List.
    1. Look for the lender from the cover page. The lender will be marked with an asterisk (*) next to their lender code. (Example: IAH* - IIT is the lender) If the lender cannot be found in the document, click View in the Request Information box
      1. Look for the lender in the Lender String field of the OCLC Information box
    2. Choose the appropriate lender in the No Ariel Lender Match box
    3. Click Attach Lender Info
  9. Requests that load into the Review section are ready to be received.
    1. Double click on a transaction number to open.
    2. Examine each page for readability and completeness.
    3. Crop, Rotate, or Deskew as necessary.
    4. Examine document as a whole, for completeness and proper page order.

More information:

This information has been updated for ILLiad 8.5.2.0.

COMPLETED.

Description:

This is a standard ILLiad queue that we do not use.

Any requests in this queue were routed here by mistake and need to be routed to their correct status.

To process:

  1. Route this request to its previous status.
    1. Click on the History tab.
    2. In the Tracking section scroll down to the bottom of the screen.
    3. Find the second oldest status in the Changed To column.
    4. Route the request to this status.
  2. Or route this request to Awaiting Extensive Searching, with a note, for further processing.

More information:

This information has been updated for ILLiad 8.5.2.0.

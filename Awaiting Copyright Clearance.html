Awaiting Copyright Clearance

Dennis Grzegorzewski, 24 November 2014 (created 3 November 2014).

COMPLETED.

Description:

This status queue holds new article requests. Before they can be processed, we must obtain permission to use and share content from the copyright holder. If copyright fees are applicable, they are attached to the request in this process.

To process:

  1. Click on the Auto Select No Problem button, in the Selection section of the Copyright ribbon.
    1. This will automatically select all requests that were published more than five years ago, and do not require copyright payment.
    2. Click the No Problem button in the Process section of the Copyright ribbon.
      1. The selected requests will be routed to the next step as detailed below.
    3. Click the OK button on the Requests Cleared pop-up window.
  2. If there are any requests remaining, they will have been published in the last five years. These requests need to be cleared one at a time.
  3. Click on the Deselect All button in the Process section of the Copyright ribbon.
    1. This will reset the search function, which is turned off by a bug when clicking the No Problem button.
  4. Select the first item in the request list.
  5. If there is a corresponding journal title match in the Journal Title list on the right side of the Copyright window,
    1. Select the lowest priced payment for that exact item from the list at the bottom of the Copyright window.
    2. Click on the Save CCC Information button in the Process section of the Copyright ribbon.
      1. This will save copyright information to the request and route it to the next step as detailed below.
    3. If the Save CCC Information button is grayed out,
      1. Move on to the next request.
  6. If there is no corresponding journal title match in the Journal Title list on the right side of the Copyright window,
    1. Move on to the next request.
  7. Repeat until all requests in the queue have been checked for copyright.
  8. Then click on the Select All button in the Selection section of the Copyright ribbon, to select all the remaining requests.
  9. Click on the No Problem button in the Process section of the Copyright ribbon.
    1. The remaining requests will be routed to the next step as detailed below.
  10. Click the OK button on the Requests Cleared pop-up window.
  11. Close the Copyright window tab.
  12. Move on to the Awaiting User Expiration Updating, Awaiting User Clearance, or Awaiting Citation Completion status queues.

More information:

Copyright cleared requests will be routed to the Awaiting Request Processing queue.

From the Awaiting Request Processing queue, requests are automatically routed...

  • to the Awaiting User Clearance status queue if the user has not been cleared,
  • to the Awaiting User Expiration Updating status queue if the user has been cleared, but the user's expiration date has passed,
  • or to the Awaiting Citation Completion status queue if the user has been cleared, and the user's expiration date has not passed.

This information has been updated for ILLiad 8.5.2.0.

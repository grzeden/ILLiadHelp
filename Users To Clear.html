Users To Clear

Dennis Grzegorzewski, 18 May 2015 (created 18 May 2015).

UNFINISHED.

Description:

This Main window System section queue holds a list of newly created user account records. These records must be verified and cleared before any requests from these users will be processed.

Student workers can check these records against Voyager to verify all the user supplied information is correct including the users expiration date. If the user is expired in Voyager they must be cleared by Staff verifying user information in Banner.

To process:

Student Worker processing using Voyager.

  1. Selecting a User to clear.

    1. In the Main window in Illiad, double-click on the Users To Clear queue to open the Users To Clear window.
    2. Double-click on a user record in the queue of Users To Clear to open a User record window.

  2. Search for the User in Voyager Circulation.

    1. Open Voyager Circulation.
    2. Click on the Patron icon, or right click on the Patron Record window and click on Search for Patron... option in the drop-down menu, if it is already open.
      1. This opens the Patron Search window.
    3. Click on the Barcode radio button.
      1. This will select searching by barcode.
    4. Copy and paste the Users ID in the Barcode: input field.
    5. Click the Find button.
      1. This will search for the patron and opens the Patron Record window, if it is found.
      2. If the patron is not found, try searching by name, or just close the user record and leave it in the queue.

  3. Verify and complete user account information.

    1. Examine each field in the User record to verify and complete the user information against the patron information in Voyager.
      1. The contents of the First Name field must exactly match the contents of the First: field in the Patron ID window in Voyager.
      2. The contents of the Last Name field must exactly match the contents of the Last: field in the Patron ID window in Voyager.
      3. The contents of the Status field must be set according to the contents of the Patron Group field in the Patron ID window in Voyager.
        1. Graduate in Illiad is Graduate Student in Voyager.
        2. Undergraduate in Illiad is Undergraduate Student in Voyager.
        3. Faculty in Illiad is Academic Employees in Voyager.
        4. Staff in Illiad is IIT Staff in Voyager.
        5. Alumni in Illiad is Alumni in Voyager.
        6. If the user is none of the above leave this in the queue for Staff to process.
      4. The contents of the Email Address field must exactly match the email address displayed in the EMail drop-down selection of the Type: field on the Address/Phone tab window in Voyager.
      5. The contents of the Expiration Date field must exactly match the contents of the Expires: field on the History tab window in Voyager.
      6. The ILLiad Authentication field is always checked.
      7. The Site field is always contains "GALVIN".
      8. The Location field always contains "ILL".
      9. The Request Limit field is always blank.
      10. The Number, Authorized Users, Fax, and Phone, fields are left unchanged.
    2. Save all changes made to the User record by clicking on the Save icon in the upper-left corner of the User record window.
    3. Close the User record window.

  4. Clear the user.

    1. Click on the Refresh icon in the upper left corner of the Illiad window to refresh the data displayed in the Users To Clear queue.
    2. Select a verified user to clear from the Users To Clear queue.
    3. If there is no matching user in the Similar Users listed on the right,
      1. Click on the Clear User button in the Clear User section of the Clear User tab ribbon.
    4. If there are any matching users in the Similar Users listed on the right,
      1. Leave this user in the queue and move on to the next.

  5. Continue until all user records have been processed.

More information:

This information has been updated for ILLiad 8.5.2.0.

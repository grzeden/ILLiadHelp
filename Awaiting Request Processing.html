Awaiting Request Processing

Dennis Grzegorzewski, 18 May 2015 (created 18 May 2015).

UNFINISHED.

Description:

All new loan requests and all copyright cleared article requests are routed to this queue. This is a standard ILLiad queue, but we have created routing rules that check the user for clearance and expiration.

Requests routed to this queue are automatically routed...

  • to the Awaiting User Clearance status queue if the user has not been cleared,
  • to the Awaiting User Expiration Updating status queue if the user has been cleared, but the user's expiration date has passed,
  • or to the Awaiting Citation Completion status queue if the user has been cleared, and the user's expiration date has not passed.

To process:

No action needed.

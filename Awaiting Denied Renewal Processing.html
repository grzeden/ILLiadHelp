Awaiting Denied Renewal Processing

Dennis Grzegorzewski, 24 November 2014 (created 3 November 2014).

COMPLETED.

Description:

This queue holds requests that have the OCLC Category NoRenewal.

The lender has denied renewal for the requests in this queue.

These requests need to have their due dates updated, and emails need to be sent to the customers to inform them that their requests can not be renewed and will need to be returned by the due date indicated.

To process:

  1. Read the Renewal Denied note in the Note section at the bottom of the request to get the new Due Date.
  2. Subtract 5 days from this date.
    1. If this date is earlier than tomorrow, use tomorrow's date.
  3. Enter this new date into the Due Date field of the Item Information section of the Borrowing Request window.
  4. Uncheck the Allow Renewals? checkbox.
  5. Click on the Save Changes icon.
  6. Click on the Send EMail icon in the User section.
  7. Click on Denied Renewal Notice in the drop-down menu.
    1. This opens the Edit EMail Form page.
  8. Click on the Send button.
  9. The request's status is automatically changed to Checked Out to Customer.
  10. The Borrowing Request window automatically closes.
  11. Repeat until all requests are processed.

More information:

If for some reason there is no Denied Renewal note, the due date can be found by doing the following from the Borrowing Request window.

  1. Click on the OCLC ribbon tab.
  2. Click on the Show icon in the Request section.
    1. This opens the OCLC Request window.
    2. Read the Due Date.
    3. Subtract 5 days from this date.
    4. Close this window.
  3. Again, if this date is earlier than tomorrow, use tomorrow's date.

This information has been updated for ILLiad 8.5.2.0.

ILLiad Help for Out of Department Processing

Dennis Grzegorzewski, 18 May 2015 (created 18 May 2015).

UNFINISHED

Description:

Listed below are the ILL processes that can be executed outside of the ILL department.

Because working from remote locations does not give access to materials and items in the department, it is limited to the processes that can be accomplished completely on the computer. As always, tackle the largest pile first.

Remote ILL Processes Available:

  • Clearing Users
  • Borrowing
    • Processing User Requests
      • Clear Copyright
      • Check User Privileges
      • Verify and Complete Citation
      • Check Galvin Catalog
      • Save PDF Files Found Online And Route to Doc Del
      • Select Lenders And Send Request
    • Receiving Requested Article Files
      • Electronic Delivery
      • Emailed PDF's
  • Lending
    • Processing Borrower Requests
      • Check Borrower Max Cost
      • Verify and Complete Citation
      • Check Galvin Catalog
      • Save PDF Files Found Online
    • Delivering Article Request PDF Files
      • Odyssey
      • Article Exchange
  • Document Delivery
    • Processing Requested Items
      • Save PDF Files Found Online
    • Delivering Article Request PDF Files
      • Odyssey
  • Updating User Expiration Dates
  • Lender Address Database Maintenance

More information:

This information has been updated for ILLiad 8.5.2.0.

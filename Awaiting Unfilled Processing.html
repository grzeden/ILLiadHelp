Awaiting Unfilled Processing

Dennis Grzegorzewski, 18 May 2015 (created 18 May 2015).

UNFINISHED.

Description:

Requests in this status queue have aged to unfilled and were not filled by any of the OCLC lenders selected. These requests need to be deleted in OCLC before they are processed in ILLiad. After which they are either cancelled, or in some cases resent to OCLC with a new selection of lenders, if available.

To process:

Student Workers - DO NOT CANCEL REQUESTS

  1. Click on the OCLC ribbon tab.
    1. This opens the OCLC ribbon.
  3. Click on the Show button in the Request ribbon section.
    1. If this opens a Save Changes? dialog box.
      1. Click on the Yes button.
      2. This closes the Save Changes? dialog box.
    1. This opens an OCLC Request window.
  6. Click on the Delete button in the Process ribbon section.
    1. This opens a Confirm Delete dialog box.
  8. Click on the Yes button.
    1. This closes the Confirm Delete dialog box, and the OCLC Request window,
    2. Changes the OCLC Status to Record not found, and
    3. Reveals the request window.

At this point the request must be canceled or resubmitted.

To cancel:

Student Workers - DO NOT CANCEL REQUESTS

  1. Click on the Borrowing Processing ribbon tab.
  2. Click on the Cancel Request button in the Next Step ribbon section.
  3. Select Your request has gone unfilled from the drop-down menu of reasons for cancellation.
  4. Check Send E-mail.
  5. Click on the Cancel Request button.
    1. The request is routed to the Cancelled by ILL Staff queue,
    2. An e-mail is sent to the patron, and
    3. The request window closes.

To resubmit:

Student Workers - DO NOT CANCEL REQUESTS

  1. Click on the Borrowing Processing ribbon tab.
  2. Copy the OCLC Number field in the OCLC Information ribbon section.
  3. Paste this in the Special Instructions field of the Item Information ribbon section.
  4. Click on the Import into Resubmission button in the System ribbon section.
    1. This opens a Confirm Import dialog box.
  6. Click on the Yes button
    1. This closes the Confirm Import dialog box,
    2. Clears the OCLC Information ribbon section, and
    3. Opens a Save Changes? dialog box.
  8. Click on the Yes button.
    1. This closes the Save Changes? dialog box.
  10. Cut the Special Instructions field of the Item Information ribbon section.
  11. Paste this in the OCLC Number field in the OCLC Information ribbon section.
  12. Click on the Refresh (F5) icon.
    1. This opens a Save Changes? dialog box.
  14. Click on the Yes button.
    1. This closes the Save Changes? dialog box.
  15. Click on the Route button in the Next Step ribbon section.
  16. Select Awaiting Request Processing (NOS Galvin) from the drop-down menu.
    1. The request is routed to the Awaiting Request Processing (NOS Galvin) queue, and
    2. The request window closes.

Repeat until all requests are processed.

More information:

Students should not cancel requests unless they have been cleared to do so.

In general, do not resubmit requests unless the request was incorrectly processed the first time, and the incorrect information has been corrected and verified in the citation for the second request.

This information has been updated for ILLiad 8.5.2.0.
